➕ 직원 추가 (대표만 가능)
🔥 Firebase 실시간 연동 ⚫ 로컬 저장
🚀 5분 셋업 — Firebase 무료로 실시간 공유하기
① Firebase 프로젝트 만들기
→ console.firebase.google.com 접속
→ 구글 로그인 후 프로젝트 추가 클릭
→ 프로젝트명 입력 (예: seowol-system)
→ Google Analytics 체크 해제 → 프로젝트 만들기
② Realtime Database 활성화
→ 왼쪽 메뉴에서 빌드 → Realtime Database
→ 데이터베이스 만들기 클릭
→ 위치: asia-southeast1 선택
→ 테스트 모드로 시작 선택 → 사용 설정
③ 웹 앱 등록 & 설정값 복사
→ 프로젝트 홈 → </> (웹) 아이콘 클릭
→ 앱 닉네임 입력 → 앱 등록
→ firebaseConfig 안의 값들을 아래에 입력
④ 보안 규칙 (사내용 간소화)
→ Realtime Database → 규칙 탭
→ 아래 내용으로 교체 후 게시:
{"rules":{".read":true,".write":true}}

* 필수 항목. 나머지는 선택 (입력하면 더 안정적)

✅ Firebase 실시간 공유 활성화됨
💡 이 HTML 파일을 카카오톡/이메일/Teams으로 직원들에게 전달하면 됩니다.
직원들이 파일을 열어 Firebase 설정값을 동일하게 입력하면 모두 같은 데이터를 실시간 공유합니다.

Excel → JSON 방식으로 일괄 등록
현재 데이터를 내보내기 후 Excel에서 편집 → 다시 가져오기
📋 순서
① 아래 "현재 데이터 내보내기" 클릭 → JSON 파일 저장
② JSON 파일을 열어 clients 배열에 업체 추가
③ 각 업체 항목 형식: {id, contractType:"기장", rep:"조용범", staff:"이수빈", name:"업체명", bizNo:"", type:"개인", fee:200000, vatFee:0, corpFee:0, source:"직접", startDate:"2025-01-01", status:"정상", note:""}
④ 저장 후 설정 → "데이터 가져오기"로 업로드
⚠️ 가져오기 시 기존 데이터가 교체됩니다. 반드시 내보내기 후 편집하세요.
